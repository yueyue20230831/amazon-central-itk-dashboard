# Amazon Central ITK Dashboard

## 🎯 项目简介
Amazon Central Team的ITK分析Dashboard，支持F2数据查看和分析。

## 🌐 在线访问
- **主页**: [ITK Dashboard](https://yueyueg.github.io/amazon-central-itk-dashboard/)

## 📊 功能特性
- ✅ ITK数据分析和可视化
- ✅ F2数据集成和报告生成
- ✅ 响应式设计，支持移动端
- ✅ 实时数据筛选和搜索
- ✅ 多维度数据展示
- ✅ ASIN链接生成和管理
- ✅ PPT报告自动生成

## 🚀 使用方法
直接访问在线地址即可使用所有功能。

## 🔧 技术栈
- HTML5 + CSS3 + JavaScript
- Chart.js 数据可视化
- Papa Parse CSV处理
- 响应式布局

## 📞 支持
如有问题请联系Amazon Central Team。
